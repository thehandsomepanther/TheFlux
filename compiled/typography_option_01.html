Typography (Montserrat and Lora)

To test out font faces, add them to the head of this document and change the <style> tag.

<link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=Lora:400,700,400italic"/> 
<link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=Patua+One"/> 
<style> 
	body { 
		font-family: 'Lora', serif; 
		font-weight:400;
	} 
	blockquote, blockquote p {
		font-family: 'Lora', serif; font-weight:100; font-style: italic;
	} 
	blockquote cite {
		font-family: 'Lora', serif; font-weight:400; font-style: normal;
	} 
	h1, h2, h3, h4, h5, h6 {
		font-family: 'Patua One', cursive; font-weight:400;
	} 
</style>
